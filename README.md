# CCGFW 自动购买脚本

## 脚本功能

CCGFW 自动购买脚本是一个基于 Tampermonkey 的油猴脚本，专为自动购买 CCGFW 的公益套餐而设计。它具有以下核心功能：

- **自动购买**：自动完成 CCGFW 公益套餐的购买流程
- **定时运行**：支持设置定时运行间隔，自动执行购买操作
- **手动触发**：可通过菜单或界面手动触发购买流程
- **多标签页防重复**：采用分布式锁机制，防止多标签页重复执行
- **自动打开网站**：可配置是否自动打开 CCGFW 网站
- **详细状态监控**：实时显示脚本运行状态、配置信息和执行历史

## 脚本特点

- **智能主备机制**：采用心跳检测和分布式锁，确保只有一个标签页作为主实例运行
- **完善的错误处理**：包含重试机制，提高购买成功率
- **友好的用户界面**：提供直观的设置界面，可实时查看脚本状态
- **灵活的配置选项**：可自定义目标套餐、定时运行间隔等参数
- **实时通知**：购买完成后发送桌面通知
- **自动跳转**：购买完成后自动跳转到用户页面并刷新
- **多套餐支持**：支持多种套餐选择，包括公益套餐和付费套餐

## 使用方法

### 基本操作

1. **访问 CCGFW 网站**：打开 [https://ccgfw.top](https://ccgfw.top)
2. **打开设置界面**：
   - 点击 Tampermonkey 扩展图标
   - 选择「CCGFW 设置」选项
   - 或在购买页面直接看到设置面板

### 配置选项

- **启用自动购买**：开启或关闭自动购买功能
- **目标套餐**：选择要购买的套餐，默认为「公益 3」
- **定时运行间隔**：设置自动运行的时间间隔（1-1440分钟）
- **自动打开网站**：设置是否自动打开 CCGFW 网站

### 手动执行

- 点击 Tampermonkey 扩展图标，选择「立即运行脚本」
- 或在设置界面点击「立即运行」按钮

### 查看状态

设置界面会实时显示以下信息：

- 当前角色（主实例/备用实例）
- 锁持有者和过期时间
- 执行中状态
- 上次成功执行时间
- 自动购买状态
- 目标套餐和定时间隔
- 下次执行时间
- 当前页面状态

## 技术实现

### 核心机制

1. **分布式锁**：使用 Tampermonkey 的存储 API 实现分布式锁，防止多标签页重复执行
2. **心跳机制**：定期发送心跳，确保主实例正常运行
3. **状态管理**：使用 Tampermonkey 的存储 API 保存配置和状态信息
4. **自动购买流程**：
   - 查找目标套餐
   - 点击购买按钮
   - 处理优惠码确认
   - 处理订单确认
   - 完成后自动跳转

## 注意事项

1. **登录状态**：使用脚本前请确保已登录 CCGFW 账号
2. **余额充足**：请确保账号余额充足，否则购买会失败
3. **网络稳定**：确保网络连接稳定，避免购买过程中网络中断
4. **浏览器兼容性**：推荐使用 Chrome、Firefox 等主流浏览器
5. **脚本更新**：定期检查脚本更新，以获取最新功能和修复

## 故障排除

### 常见问题

1. **脚本不运行**：
   - 检查 Tampermonkey 是否已启用
   - 检查脚本是否已启用
   - 刷新页面后重试

2. **购买失败**：
   - 检查账号余额是否充足
   - 检查网络连接是否稳定
   - 手动尝试购买，确认网站是否正常

3. **多标签页冲突**：
   - 脚本会自动处理多标签页冲突，确保只有一个标签页执行购买
   - 如果出现问题，可关闭所有标签页后重新打开
